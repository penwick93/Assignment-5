Angular Introduction

An introduction to Angular using the ToDo list example

Angular is quickly becoming one of the most useful tools in a programer's toolbox.

AngularJS is known as a responsive programing language. This type of language simply means that when interacting with the various buttons, tabs and elements within the language, the entire webpage will not reload but simply respond to the touch of the mouse and load the requested information. This is great for single page applications. In this example, we will cover some of the basics of Angular.js that were used in creating the ToDo application.

To begin, we must understand the Angular basics that go into creating this application; directives, controllers and scopes. Directives are used to extend HTML and are denoted at the ng-directive. Some examples are shown below:

Directives directly bind the data to the Angular JS application and initializes application variables. Controllers are used as Javascript objects and control the data of AngularJS applications. Controllers are used often with the Scope function of AngularJS. Scope, or $scope, will invoke the controller with a scope object. It is the binding part between the HTMl view and the JavaScript controller. An example of controllers and scopes are below:
